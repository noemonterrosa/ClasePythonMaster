Para el proyecto2 de Python los archivos que se crearon son los siguientes:
- Crear_BD_AWS.py: Script de Python para crear la BD "stocks" en AWS
- CrearTablas.py: Script de python para leer los archivos csv, transformar los datos, crear las tablas y cargar los datos de las acciones
- Analisis.py: Script de python para el analisis de los datos, leyendolos directamente de la BD
- sql_queries.py: Archivo DDL para crear las tablas de la BD
- Proyecto2.ipynb: Archivo borrador de Jupyter Notebook que sirvio para ir desarrollando el codigo de carga y transformacion
- Proyecto2_2.ipynb : Archivo borrador de Jupyter Notebook que sirvio para desarrollar el analisis de los datos
